    Text Fields

    Text fields components are used for collecting user provided information.

    All the properties which are not in the API list will be directly forwarded to the input element (such as maxlength).

    API

    Props

    Prop Default Description
    value string Value of the text input.
    color string primary The color of the input when active.
    filled boolean false Changes the variant of the input to filled.
    solo boolean false Changes the variant of the input to solo.
    outlined boolean false Changes the variant of the input to outlined.
    flat boolean false Removes any shadow from the input.
    dense boolean false Reduces the input height.
    rounded boolean false Adds a border radius to the input.
    clearable boolean false Add input clear functionality.
    readonly boolean false Puts input in readonly state.
    disabled boolean false Disable the input.
    placeholder object null The input placeholder content.
    hint string Hint text.
    counter number false Creates counter for input length.
    rules Array undefined An array of functions which take input value as arguement and return error message.
    validateOnBlur boolean false Delays validation till blur.
    error boolean false Error state of the input.
    id any undefined Id of the text input.

    Events

    Event Returns Description
    focus DOMEvent Slot for prepend inside the input.
    blur DOMEvent DOM Event
    input DOMEvent DOM Event
    change DOMEvent DOM Event

    Slots

    Slot Slot Props Description
    prepend-outer None Slot for prepend outside the input.
    prepend None Slot for prepend inside the input.
    default None Slot for prepend inside the input.
    content None Slot for prepend inside the input.
    clear-icon None Slot for the icon when `clearable` is true.
    append None Slot for append inside the input.
    append-outer None Slot for append outside the input.

    Usage

    Examples

    Dense

    You can reduces the text field height with dense prop.

    Disabled and Readonly

    Text fields can be disabled or readonly.

    Icons

    You can add icons to the text field with prepend, append, prepend-outer and append-outer slots.

    Clearable

    When clearable, you can customize the clear icon with clear-icon.



    Counter

    Use a counter prop to inform a user of the character limit. The counter does not perform any validation by itself. You will need to pair it with either the internal validation system, or a 3rd party library.

    Password

    A password input can be used with an appended icon and callback to control the visibility.

    Validation

    Svelte Materialify provides inbuild simple validation through the rules prop. The prop accepts a array of callbacks with the input value as an arguement. By default this runs a check whenever the user types something, to delay the validation till the input loses focus use validateOnBlur.

    const rules = [
  (value) => {
    if (condition) return 'Error Message';
    return false;
  },
];
